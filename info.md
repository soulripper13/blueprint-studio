# Blueprint Studio

A modern, feature-rich file editor for Home Assistant with a VS Code-like experience.

## Key Features

- **Modern Dark UI** - VS Code-inspired interface
- **Multi-tab Editing** - Open multiple files at once
- **File Tree Explorer** - Navigate your config with ease
- **Syntax Highlighting** - YAML, JSON, Python, JS, and more
- **YAML Validation** - Real-time error detection
- **Keyboard Shortcuts** - Ctrl+S to save, Ctrl+F to search
